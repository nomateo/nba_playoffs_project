# nba_playoffs_project

# статистика плей-офф НБА 2010–2024

Разведочный анализ реального датасета box scores плей-офф за 14 сезонов.

## Вопросы исследования
- Кто доминировал в плей-офф за 14 лет?
- Как распределена результативность за игру?
- Дошла ли 'трёхочковая революция' до плей-офф?
- Какие показатели бокс-скора связаны между собой?
- Как минуты влияют на очки?

## Результаты
- Леброн Джеймс - лидер по суммарным очкам (6.134)
- Лука Дончич - лидер по среднему за игру (31.6 PPG)
- 3-очковых попыток в плей-офф стало вдвое больше за 14 лет (с 18 до 38)
- Корреляция минут и очков: 0.76

## Стек
Python + pandas + numpy + seaborn + matplotlib
